კოვიდ-19 ვაქცინაცია თბილისში

რუკის შესახებ

რუკა ასახავს თბილისში არსებულ ვაქცინაციის ცენტრებსა და საზოგადოებრივი ტრანსპორტის აკრძალვის პირობებში, მათი მიმართულებით დანიშნული ავტობუსების მარშრუტს. მარჯვენა ზედა კუთხეში არსებული ფუნქციებით, შესაძლებელია ვაქცინის სახეობისა და მარშრუტების ფენის გააქტიურება/გამორთვა. რუკაზე დატანილ მონაცემებზე მაუსის გადატარებით მიიღებთ ინფორმაციას ვაქცინაციის ცენტრების მიერ შემოთავაზებული აცრის, ცენტრის დასახელებისა და მისამართის შესახებ.
ავტობუსების მარშრუტებისა და სამუშაო პერიოდის შესახებ ინფორმაციისთვის გადაატარეთ მაუსი ხაზოვან მონაცემებს.
